# FluidImage

### autor:

[Alexander Nilsson](mailto:alexander.nilsson@creuna.no)

### dependencies:

- `classnames`
- `prop-types`

### usage:

```js
<FluidImage
  alt="My beautiful clownfish"
  className="my-image"
  fallback="fallback-image.jpg"
  focusPoint={{
    x: 19.4,
    y: 82.3
  }}
  src="clownfish.jpg"
/>
```
